# Варианты цветовых схем для светлой темы

На основе веб-безопасных цветов с [artlebedev.com/tools/colors](https://www.artlebedev.com/tools/colors/)

## Вариант 1: Классический деловой (Рекомендуется)

**Особенности:** Сдержанная, профессиональная палитра, подходит для финансовых приложений

```css
:root {
  /* Фоны - серые веб-безопасные */
  --bg-primary: #CCCCCC;        /* Светло-серый фон */
  --bg-secondary: #FFFFFF;       /* Белый */
  --bg-tertiary: #CCCCCC;        /* Светло-серый для альтернативных строк */
  
  /* Текст - темно-серые */
  --text-primary: #333333;       /* Темно-серый основной текст */
  --text-secondary: #666666;     /* Серый вторичный текст */
  
  /* Границы */
  --border-color: #CCCCCC;       /* Светло-серый */
  --border-color-strong: #999999; /* Темно-серый */
  --input-border: #999999;       /* Темно-серый для полей */
  
  /* Hover эффекты - голубые */
  --hover-bg: #CCFFFF;           /* Очень светло-голубой */
  --hover-bg-strong: #99CCFF;    /* Светло-голубой */
  
  /* Заголовок таблицы */
  --table-header-bg: linear-gradient(180deg, #CCCCCC 0%, #999999 100%);
  
  /* Основные акценты */
  --primary-color: #3366FF;      /* Синий для кнопок */
  --primary-hover: #0033CC;      /* Темно-синий */
  --danger-color: #CC0000;       /* Красный для опасных действий */
  --danger-hover: #990000;       /* Темно-красный */
  
  /* Тени */
  --shadow: rgba(0, 0, 0, 0.15);
  --shadow-hover: rgba(0, 0, 0, 0.25);
}
```

**Кнопки:**
- Primary: `#3366FF` → `#0033CC` (hover)
- Danger: `#CC0000` → `#990000` (hover)

## Вариант 5: Минималистичный светлый

**Особенности:** Максимально светлая палитра, минимальные акценты

```css
:root {
  /* Фоны - очень светлые */
  --bg-primary: #CCCCCC;         /* Светло-серый */
  --bg-secondary: #FFFFFF;       /* Белый */
  --bg-tertiary: #CCCCCC;        /* Светло-серый */
  
  /* Текст */
  --text-primary: #333333;       /* Темно-серый */
  --text-secondary: #999999;     /* Серый */
  
  /* Границы - едва заметные */
  --border-color: #CCCCCC;       /* Светло-серый */
  --border-color-strong: #999999; /* Серый */
  --input-border: #CCCCCC;       /* Светло-серый */
  
  /* Hover эффекты - очень легкие */
  --hover-bg: #CCFFFF;           /* Очень светло-голубой */
  --hover-bg-strong: #CCFFFF;    /* Очень светло-голубой */
  
  /* Заголовок таблицы */
  --table-header-bg: linear-gradient(180deg, #CCCCCC 0%, #CCCCCC 100%);
  
  /* Основные акценты - умеренные */
  --primary-color: #6699FF;      /* Светло-синий */
  --primary-hover: #3366FF;      /* Синий */
  --danger-color: #FF6666;       /* Светло-красный */
  --danger-hover: #FF3333;       /* Красный */
  
  /* Тени - легкие */
  --shadow: rgba(0, 0, 0, 0.08);
  --shadow-hover: rgba(0, 0, 0, 0.12);
}
```

**Кнопки:**
- Primary: `#6699FF` → `#3366FF` (hover)
- Danger: `#FF6666` → `#FF3333` (hover)

## Рекомендация

**Для финансового приложения рекомендую Вариант 1 (Классический деловой):**
- ✅ Профессиональный вид
- ✅ Хорошая читаемость
- ✅ Универсальность
- ✅ Стандартные цвета для бизнес-приложений
- ✅ Достаточный контраст

**Альтернативно можно использовать Вариант 2 (Мягкий современный)** если нужен более современный и свежий вид.
